# Code-Generators
生成代码生成器的生成器

## 项目介绍
基于React+Spring-Boot+Picocli+对象存储的代码生成器共享平台，开发者可以在平台上制作并发布代码生成器，用户可以搜索、下载、在线使用代码生成器，管理员可以集中管理所有代码生成器

## 名词解释
1. 源代码：用户上传的代码
2. 源代码模板：用于制作目标代码
3. 目标代码：用户最终需要的代码
4. 生成器源代码：用于测试生成的逻辑
5. 生成器模板代码：用于制作生成器的模板
6. 生成器目标代码：通过生成器模板生成出来的用于制作目标代码的生成器
